# LSTM_PERU_RUSLE
 Predicción de erosión hídrica en la cuenca del río Lurín  Integración de una red neuronal  LSTM multihorizonte con el modelo espacial **RUSLE** para proyectar la precipitación mensual, estimar el factor de erosividad de la lluvia (R) y analizar la pérdida potencial de suelo en la cuenca del río Lurín, Perú
